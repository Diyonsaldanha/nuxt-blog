When deciding how to build my portfolio, I wanted something fast, SEO-friendly, and enjoyable to write. I've been working with Vue, but setting up routing and SSR manually can be a headache. 

Enter **Nuxt 3**.

### The Best Parts

1. **Auto-imports:** I don't have to write `import { ref } from 'vue'` in every single file.
2. **File-based routing:** Dropping a `.vue` file into the `pages` directory and instantly having a route is a massive timesaver.
3. **Nuxt Content:** Writing this exact blog post in a Markdown file and having Nuxt render it beautifully without a database is incredibly satisfying.
